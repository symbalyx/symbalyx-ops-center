🛠️ Script — Site WordPress avec failles (scénario typique)
🎯 Correctif 99€3 min d'appel
1️⃣ OUVERTURE (5 sec) Vous : "Bonjour [Nom], c'est Arsène de Symbalyx. Je suis passé sur votre site WordPress, j'ai vu 3 failles de sécurité. Je peux vous les montrer en 30 secondes ?"
→ Taux d'acceptation ~80%
2️⃣ DÉMO — Les erreurs concrètes Vous : "J'ai noté 3 choses : 1. Vos identifiants admin fuient — en tapant /?author=1 sur votre site, je vois le login admin. Un hacker peut bruteforcer votre mot de passe. Avec un robot, c'est 1000 tentatives/minute. Risque : piratage total du site. 2. Votre debug.log est exposé — dans /wp-content/debug.log, je vois les erreurs PHP de votre site. Ça donne le chemin du serveur, la version de PHP, des tentatives de connexion SQL. Risque : un hacker peut cartographier votre infra. 3. Votre version WP est visible — les hackers savent exactement quelles failles CVE tester. Votre version a [X] failles connues. Risque : infection par malware, site blacklisté Google." 3️⃣ VENTE (15 sec) Vous : "Je corrige les 3 failles aujourd'hui pour 99€, et je vous offre un audit complet de votre site. Ça vous va ?" → Si oui → encaisses 99€. Si non → envoie correction.html par SMS.
